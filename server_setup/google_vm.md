# Create a always free virtual machine at google
1. Go to [](console.cloud.google.com)
2. Create a VM
3. Choose the region: us-west1, us-central1 or us-east1
4. Choose the zone: Any
5. Choose the machine series: E2
6. Choose the machine type: e2-micro (2 vCPU, 1 GB memory)
7. Connect to the VM using SSH-in-browser
8. Follow the steps in [Setup on Debian-based Linux](debian.md)
